### 지도학습과 비지도학습

KNN 알고리즘을 통한 학습이 정확도 100%인 이유가 뭘까?  

머신러닝 알고리즘:  
* 강화학습
* 비지도학습
    - 군집화
    - 변환
    - 연관

* 지도학습  
    - 분류: 비연속적인 결과, 선택한 데이터가 어떤 그룹에 속하는지 예측하는 모델. **그룹에 대한 값**
    - 회귀: 연속적인 결과, 선택한 데이터의 결괏값에 가까운 실수값을 예측하는 모델. **연속성이 있는 값**


    지도학습은 데이터와 정답을 함께 알려주고, 비지도 학습은 정답을 알려주지 않는다. 

    + 훈련 데이터
        + 입력: 데이터
        + 타킷: 정답  
        (1주차에서 사용했던 물고기의 길이와 무게는 **특성**이라고 한다.)
### 훈련세트와 테스트세트

정확도 100%의 문제점
: 훈련에 사용한 데이터와 평가하는 데이터가 같기 때문에 100%인 것!

&rarr; *신뢰할 수 있는 모델은 어떻게 만들까?*  
데이터 세트를 분리해서 **훈련세트**와 **테스트 세트**를 만들기

하나의 물고기 데이터: *샘플* (입력(길이 특성 + 무게특성) + 타깃)

### 샘플링 편향
훈련세트와 테스트 세트에 샘플이 골고루 섞이지 않고 한쪽으로 샘플이 치우치는 것
&rarr; *해결방법*: 테스트 세트에 샘플이 골고루 섞이면 된다

```
np.random.seed(42)
```

### 데이터 전처리
numpy와 사이킷런을 통해 훈련세트와 테스트세트를 나누고 샘플링 편향을 해결,

**+스케일을 맞춰서 최근접 이웃들의 위치가 잘 맞도록 하는 과정이 필요하다**

##### 표준점수
표준점수 = (데이터 값- 평균) / 표준편차  

평균과 표준편차 구하기
```
mean=np.mean(train_input, axis=0)
std=np.std(train_input, axis=0)
```
*새로운 샘플도 표준점수로 변환해주는 것을 잊지 말아야 한다*
