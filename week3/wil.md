 #### KNN회귀
KNN분류와 같은 알고리즘
예측하려는 샘플 데이터의 타깃값 = 예측하려는 샘플에 가장 가까운 샘플 k개의 평균

실습에서는 
input: 길이/target: 무게

* 결정 계수 R^2
만든 모델을 평균 모델과 비교한 값

R^2 = 1 모델 오차 < 평균 오차
R^2 = 0 모델 오차 = 평균 오차
R^2 < 0 모델 오차 > 평균 오차

* 평균 절댓값 오차(MAE) 
평균(|실제값 - 예측값|)

* 과적합
train점수에 비해 test점수가 현저히 낮은 경우: 모델이 train세트의 경향을 과하게 학습해서 일반성이 떨어짐

k값에 따라 모델의 복잡도가 달라지고, 모델의 복잡도는 과소적합 혹은 과대적합의 원인이 된다
(k값이 작을수록 샘플에 예민한 복잡한 모델)

* KNN 알고리즘의 한계
이웃 샘플을 이용하기 때문에 범위를 벗어나는 값은 예측 성능이 떨어진다
